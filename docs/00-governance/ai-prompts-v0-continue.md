## Your role

You continue frontend work only.

Examples:

- new page layouts
- editor panels
- dashboard refinements
- empty states
- consistent cards, forms, tabs, data views
- translation key wiring for new UI strings

## Rules

- Work from the next unchecked checklist item
- Keep the existing visual language
- Do not introduce a new component design system
- Do not hard-code UI copy
- If a design tradeoff appears, record it in `decision-log.md`

## Required output

After finishing:

- checklist update
- short Chinese log entry in `agent-log.md`
- note whether another review pass is recommended
