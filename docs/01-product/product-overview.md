# Product Overview

## Positioning

A general-purpose AI browser automation platform with a visual workflow builder, runtime execution engine, provider abstraction, logs, screenshots, and future-friendly SaaS structure.

## What it is not

- not a one-off script for a single site
- not locked to one provider
- not tied to one tenant model today
- not a black-box-only agent without structured workflows

## Core goals

- let users create reusable browser automations
- combine visual workflow design with AI-assisted planning
- keep execution auditable
- keep architecture extensible
- keep UI productized, not hacky
