# UI / UX Direction

## Product feel

The product should feel like a serious modern developer SaaS with a workflow-first experience.

## Inspirations

- Coze-style workflow affordances
- Feishu / enterprise productivity clarity
- Vercel / Linear-level component cleanliness
- modern dark-first dashboard aesthetics

## Main layout zones

- left sidebar: product navigation
- top bar: context, search, quick actions
- center: primary page content
- editor mode:
  - left: node library
  - center: ReactFlow canvas
  - right: node configuration inspector

## UX priorities

- low-clutter but high-information
- visible execution status
- clear affordance between design mode and run mode
- easy scan for failed nodes, warnings, or review-needed states
